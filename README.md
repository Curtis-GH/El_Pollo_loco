# El_Pollo_loco# El Pollo Loco

Ein Browser-basiertes Jump-and-Run-Spiel, entwickelt im Rahmen der Developer Akademie. Der Spieler steuert den Charakter Pepe durch eine Wüstenlandschaft, sammelt Coins und Flaschen, besiegt Hühner-Gegner und muss sich am Ende einem Endboss stellen.

## Tech-Stack
- Vanilla JavaScript (OOP, Klassenstruktur)
- HTML5 Canvas
- CSS3

## Features
- Charakteranimationen (Idle, Sleep, Walk, Jump, Hurt, Dead)
- Zwei Gegnertypen + Endboss mit steigender Geschwindigkeit
- Sammelsystem für Coins und Flaschen mit Statusbars
- Mobile Touch-Steuerung
- Sound- und Musiksteuerung mit Lautstärkeregler
- Responsive Design für Desktop und Mobile

## Steuerung
- **Pfeiltasten links/rechts** – Bewegen
- **Leertaste** – Springen
- **D** – Flasche werfen
